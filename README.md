# Subwoofer Array Designer (Web)

## Rodar local (macOS/Windows)
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Publicar online (Streamlit Cloud)
1. Crie um repositório no GitHub e envie estes arquivos.
2. Vá para https://share.streamlit.io e crie um novo app apontando para `streamlit_app.py`.
3. O serviço gera uma URL pública automaticamente.
