# SubArrayDesigner-WebApp-NewDevice — v3.3
Agora com **gestão de Fabricantes** e **Modelos de caixas** dentro do app (Catálogo → Admin).

## Novidades
- Adicionar fabricante/modelo via formulário (com drivers, tipo, diretividade, faixa, potência, sensibilidade, SPL máx., links).
- Filtrar por fabricante/modelo.

- Importar e baixar **models.json** direto no app.

## Deploy (Streamlit Cloud)
- Repository: `angelomacae-rgb/SubArrayDesigner-WebApp-NewDevice`
- Branch: `main`

## Rodar local

pip install -r requirements.txt
